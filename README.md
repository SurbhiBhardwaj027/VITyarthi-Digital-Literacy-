# VITyarthi-Digital-Literacy-

Digital literacy is the ability to confidently find, evaluate, create, and communicate information using technology, ranging from computers to smartphones. It combines technical skills (using tools) with cognitive skills (critical thinking) to navigate digital platforms safely, responsibly, and effectively in daily life, work, and education.
